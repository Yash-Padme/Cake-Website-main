## 📖 About the Project

This is a **freelance project** developed for **Ayushi's Cake**, a local cake bakery. The goal was to give the bakery a professional digital presence with a fully responsive website that allows customers to browse the menu, explore cake options, and place orders directly via WhatsApp — increasing customer engagement and driving online orders.

---

## ✨ Features

- 🎠 **Hero Slider** — Eye-catching animated carousel showcasing Birthday, Wedding, Dessert, and Pastry cakes
- 🍰 **Featured Menu** — Highlighted cake categories with images and descriptions
- 📖 **About Section** — Bakery story and key statistics (counter animations)
- 🖼️ **Gallery** — Visual showcase of cake creations
- 📋 **Menu Section** — Full catalogue of available cakes and flavours
- ⭐ **Testimonials** — Customer reviews and feedback carousel
- 👨‍🍳 **Chef Section** — Meet the baking team
- 📦 **Online Order Form** — Place custom cake orders (flavour, weight, message, delivery date/time, price range)
- 📲 **WhatsApp Integration** — Orders are sent directly to the bakery's WhatsApp
- 📰 **Blog Section** — Cake tips and bakery news
- 📱 **Fully Responsive** — Optimised for mobile, tablet, and desktop

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and markup |

| Font Awesome / Ionicons / Flaticon / IcoMoon | Icon sets |

## 🚀 Getting Started

No build tools or dependencies required — this is a static website.

1. **Clone the repository**
   ```bash
   git clone https://github.com/Yash-Padme/Cake-Website-main.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Cake-Website-main/Cake-Website-main
   ```

3. **Open in browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` to open it in your default browser.

## 📲 WhatsApp Ordering Integration

When a customer fills out the **Order a Cake** form and submits it, their order details (name, email, address, phone, date, time, flavour, weight, message, and price range) are automatically formatted and sent as a pre-filled WhatsApp message to the bakery's number — making order management seamless without a backend.
